# 💸 Personal Expense Tracker

A simple and interactive **Personal Expense Tracker** built using **Python, Pandas, and Streamlit**.
This application helps users track, analyze, and visualize their daily expenses in a clean dashboard.

---

## 🚀 Live Demo

👉 https://expense-tracker-daily-purpose-app.streamlit.app

---

## 📌 Features

* ➕ Add new expenses (date, category, amount, description)
* 📅 Filter expenses by date range
* 🗂️ Filter by categories
* 📊 Interactive dashboard with key metrics:

  * Total spending
  * Number of transactions
  * Category count
* 📈 Visualizations:

  * Category-wise spending (bar chart)
  * Monthly trend (line chart)
  * Spending distribution (pie chart)
* 📥 Download filtered data as CSV
* 🎨 Clean and user-friendly UI using Streamlit

## ⚙️ Installation & Setup (Run Locally)

1. Clone the repository:

```
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
